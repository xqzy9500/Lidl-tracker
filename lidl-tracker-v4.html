1 Notion Setup

How to get your Notion API key & Database IDs
1. Create integration: Go to notion.so/my-integrations → New integration → Copy the API key.

2. Create 3 Notion databases: "Free Fruit", "Free Vegetable", "Free Bakery". Each needs columns: Name (title) and Expiry Date (date).

3. Share each database with your integration (click ••• → Connect to → your integration).

4. Get Database ID: Open each database in browser — copy the ID from the URL between the last / and the ?
